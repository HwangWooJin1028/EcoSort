# EcoSort
Yolov5 기반 실시간 쓰레기 분류 프로그램, EcoSort
<br/><br/><br/>


## 개요
- Yolov5 모델 기반으로 웹캠을 이용하여 실시간으로 쓰레기를 분류한다
- 또한, 사용자가 업로드한 사진 파일을 이용하여 쓰레기를 분류한다.
- 분류한 쓰레기는 label을 통해 보여준다
